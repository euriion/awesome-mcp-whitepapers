# Awesome MCP Whitepapers [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> A curated list of whitepapers, tools, resources, and documentation for the Model Context Protocol (MCP)

The Model Context Protocol (MCP) is an open protocol that enables seamless integration between LLM applications and external data sources and tools. This repository collects whitepapers, technical documentation, tool implementations, and community resources.

## Tools and Implementations

### MCP Servers

MCP servers provide context and tools to LLM applications:

- [Filesystem Server](https://github.com/modelcontextprotocol/servers/tree/main/src/filesystem) - Access and manipulate local files
- [GitHub Server](https://github.com/modelcontextprotocol/servers/tree/main/src/github) - Interact with GitHub repositories
- [Google Drive Server](https://github.com/modelcontextprotocol/servers/tree/main/src/gdrive) - Access Google Drive files
- [PostgreSQL Server](https://github.com/modelcontextprotocol/servers/tree/main/src/postgres) - Query PostgreSQL databases
- [Puppeteer Server](https://github.com/modelcontextprotocol/servers/tree/main/src/puppeteer) - Browser automation
- [Slack Server](https://github.com/modelcontextprotocol/servers/tree/main/src/slack) - Interact with Slack workspaces

### MCP Clients

Applications that consume MCP servers:

- [Claude Desktop](https://claude.ai/download) - Anthropic's desktop application with MCP support
- [MCP Inspector](https://github.com/modelcontextprotocol/inspector) - Development and debugging tool for MCP servers
